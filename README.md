# Visual Canvas Exploration

ブラウザ上で2D・3Dグラフィック表現の様々な可能性を探索するサイト。

## 🚀 はじめに

このプロジェクトは、WebGL, Three.js, Canvas API, GSAP などの最新ウェブ技術を駆使した視覚的実験のギャラリーです。

### 開発環境の実行

```bash
npm install
npm run dev
```

### ビルド

```bash
npm run build
```

## 📂 ディレクトリ構造

- `src/labs/`: 各グラフィック実験（2D/3Dデモ）のソースコード。
- `src/components/`: UIコンポーネント。
- `src/styles/`: グローバルスタイル（index.css）。
- `.github/workflows/`: GitHub Pages への自動デプロイ設定。

## 🛠 技術スタック

- **Vite**: ビルドツール & 開発サーバー
- **Three.js**: 3Dグラフィックス API
- **GSAP**: 高性能なアニメーションライブラリ
- **Canvas API**: 2Dグラフィックス
