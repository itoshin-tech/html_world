# プロジェクト仕様書：Visual Canvas Exploration (仮)

## 1. プロジェクト概要
ブラウザ上で実現可能な2Dおよび3Dグラフィックの表現力を追求し、その可能性を提示するギャラリー兼ショーケースサイトを制作します。

## 2. コンセプト
- **探索と発見**: 最新のウェブ技術（WebGL, WebGPU, SVG, CSS 3D等）を駆使した視覚表現の実験場。
- **インタラクティブ性**: ユーザーがパラメータを操作し、リアルタイムで変化を楽しめる体験。
- **圧倒的な美学**: 高品質なデザインとスムーズなアニメーションによる、プレミアムなユーザー体験（UX）。

## 3. 主要機能
### 3.1. グラフィック・ギャラリー
- 各種技術（Three.js, Canvas API2D, Shader等）を用いたデモのリスト表示。
- カテゴリ別（2D, 3D, Generative, Physics等）のフィルタリング機能。

### 3.2. インタラクティブ・サンドボックス
- デモ画面上でのパラメータ調整用UI（GUIコントローラー）。
- ソースコードの一部表示、または技術解説の表示。

### 3.3. 技術ドキュメント
- 各表現で活用している数学的アルゴリズムやライブラリの簡潔な解説。

## 4. 技術スタック
### 4.1. フロントエンド
- **コア**: HTML5, Vanilla JavaScript (ESNext)
- **3Dグラフィックス**: Three.js, OGL, または WebGL 原生 API
- **2Dグラフィックス**: Canvas API, SVG, GSAP (アニメーション)
- **スタイリング**: Vanilla CSS (最新のCSS機能を活用)
- **ビルドツール**: Vite (高速な開発体験)

## 5. デザイン方針
- **ダークモード中心**: グラフィックが映えるよう、深く洗練されたダークテーマを採用。
- **グラスモーフィズム**: 半透明のパネルやぼかし効果を用いた、奥行きのあるUI。
- **モダン・タイポグラフィ**: `Outfit` や `Inter` などの幾何学的なフォントを使用。
- **マイクロ・アニメーション**: ホバー、スクロール、遷移時の滑らかな動き。

## 7. GitHub Pages デプロイ仕様
本プロジェクトは GitHub Pages を利用して公開します。ビルドプロセスとデプロイの設定を最適化します。

### 7.1. デプロイ構成
- **ソース管理**: GitHub
- **CI/CD**: GitHub Actions を使用した自動デプロイ。
- **公開ディレクトリ**: `docs` フォルダ、または `gh-pages` ブランチ。本プロジェクトでは Vite のビルド成果物を `dist` に出力し、それをデプロイします。

### 7.2. 設定の注意点
- **ベースパス**: GitHub Pages の URL（`https://<username>.github.io/<repository>/`）に対応するため、Vite の設定ファイル `vite.config.js` で `base` パスを設定します。
- **ルーティング**: シングルページアプリケーション（SPA）構成の場合、GitHub Pages の fallback 設定（404.htmlのリダイレクトなど）を検討します。

```text

```

## 9. フェーズ1の開発目標
1. プロジェクトの基礎構造の構築 (Vite + Vanilla JS)
2. **GitHub Actions による自動デプロイ環境の構築。**
3. ヒーローセクションへの印象的な3D背景の実装。
4. 2D Canvas を用いたパーティクルアニメーションデモの追加。
5. Three.js を用いたシンプルな3Dオブジェクト操作デモの追加。
